# 🧠 EvoBrain - RL + Neuroevolution 24/7

> Sistema de previsão contínua que aprende com erros e evolui em tempo real.
> **Alternativa open-source superior ao MiroFish.**

## 🚀 O Que o EvoBrain Faz

- **Aprendizado contínuo 24/7** - Nunca para de aprender
- **RL + Neuroevolution** - 1000+ agentes evoluindo geneticamente
- **API externa em tempo real** - Dados frescos sempre
- **Custo zero** - Inferência local, sem API de LLM
- **Aprende com erros** - Feedback loop que corrige agentes

## 🆚 Comparação com MiroFish

| Funcionalidade | EvoBrain | MiroFish |
|----------------|----------|----------|
| Motor | RL + PyTorch | LLM (API cara) |
| Aprendizado | Contínuo (24/7) | Estático |
| Custo | $0 | $$ por chamada |
| Dados | API externa real-time | Upload manual |
| Evolução | Neuroevolution + crossover | ❌ |
| Escala | Milhões de agentes | Limitado por budget |

## 📦 Instalação

```bash
git clone https://github.com/Gabrie50/EvoBrain.git
cd EvoBrain
pip install -r requirements.txt
```
